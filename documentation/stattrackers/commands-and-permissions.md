| Command                                    | Description                 | Permission                          |
|--------------------------------------------|-----------------------------|-------------------------------------|
| `/stattrackers`                            | Opens the StatTrackers GUI  | `stattrackers.command.stattrackers` |
| `/stattrackers give <player> <tracker_id>` | Give a player a StatTracker | `stattrackers.command.give`         |
| `/stattrackers reload`                     | Reload the plugin           | `stattrackers.command.reload`       |
